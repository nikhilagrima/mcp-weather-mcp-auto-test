# weather-mcp-auto-test

MCP Server deployed via AI MCP Deployment Platform

## Files

- `server.py` - MCP server code
- `requirements.txt` - Python dependencies
- `Procfile` - Railway start command
- `railway.json` - Railway configuration

## Environment Variables

None

## Deploy to Railway

Click the button below to deploy this MCP server to Railway:

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/nikhilagrima/mcp-weather-mcp-auto-test)

Or manually:
1. Go to https://railway.app/new
2. Select "Deploy from GitHub repo"
3. Choose `nikhilagrima/mcp-weather-mcp-auto-test`
4. Add environment variables if needed
5. Deploy!

## MCP Endpoint

After deployment, your MCP endpoint will be:
`https://your-app.up.railway.app/mcp`
